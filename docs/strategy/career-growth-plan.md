## The Strategy: AI Security Specialization

### Why AI Security
- Companies are deploying AI everywhere but nobody knows how to secure it
- Prompt injection, data poisoning, model theft, PII leakage are real unsolved problems
- Almost zero people specialize in this yet
- Combines everything already in the toolkit: coding, security certs, AI knowledge
- AI Red Teaming / AI Security roles pay $160-200k+
- OWASP released the LLM Top 10 - almost nobody knows this stuff yet

### Target Roles (no clearance required)
- **AI Security Engineer** - securing LLM deployments, building guardrails
- **AppSec Engineer** - code review for vulnerabilities, SAST/DAST tools, threat modeling
- **Product Security Engineer** - Google, Microsoft, Stripe all hire green card holders
- **Cloud Security Engineer** - AWS-heavy roles
- **DevSecOps Engineer** - bridge role between current skills and security

### Companies That Hire Green Card Holders
- Big tech (Google, Microsoft, Meta, Apple)
- Fintech (Stripe, Capital One, JPMorgan)
- SaaS / healthtech / any non-classified private sector
- Target **remote roles** to access SF/NY salary bands from Charleston

### 1. PWPA - Practical Web Pentest Associate (NOW - FREE)
- **Free right now** - no reason not to grab this immediately
- Web app pentesting directly relevant to AppSec roles
- Already completed the START HERE module on TCM
- Practical exam (actually hack a web app, not multiple choice)

### 2. AWS Solutions Architect Associate (ASAP - No AWS certs yet)
- **Company (g/d/n/a) pays 100%** of exam fees + $500 bounty on pass
- Must get this first - it's the AWS foundation, Security Specialty builds on it
- Satisfies company requirement of 1 cert/year for good standing
- Study in parallel with GSEC

### 3. AWS Security Specialty (After SA Associate)
- **Company pays 100%** of exam fees + $1,000 bounty on pass
- Directly aligned with AI Security career path
- Requires solid AWS knowledge (that's why SA Associate comes first)

### 4. GSEC (In Progress)
- GIAC cert, carries serious weight with employers
- Keep on current timeline, don't let AWS derail this

### 5. GCIH (~3 months out)
- Incident handling cert, essential for security roles
- Combined with GSEC creates a strong GIAC foundation

### 6. PAPA - Practical AI Pentest Associate (THE NICHE CERT)
- TCM's AI security certification - this IS the differentiator
- Hands-on exam: 2 days to pentest an AI/LLM application + 2 days to write report
- Comes with 12 months access to AI Hacking 101 course
- Tests: prompt injection, AI chatbot exploitation, LLM vulnerabilities
- A resume with GSEC + GCIH + PWPA + PAPA says: "I do security AND I hack AI systems"
- Almost nobody has this cert yet - early mover advantage

### 7. PNPT - Practical Network Penetration Tester (After GCIH)
- TCM's flagship cert, gaining real industry recognition
- ~$400, way cheaper than GIAC
- Practical exam (hack a network, write a report)
- GIAC proves you know the theory, PNPT proves you can do it
- Hiring managers love that combo

### 8. TCM Security Courses (Already Paid - Finish These)
- **Practical Ethical Hacking** - finish first, most valuable for the path
- **Python 101 for Hackers** - solves the Python gap in a security context
- **Practical Bug Bounty** - good for AppSec interview stories
- **AI Hacking 101** - prepares for PAPA cert, this IS the niche
- Skip Linux/Windows privesc and OSINT for now (nice-to-haves)

### AI Fundamentals to Brush Up On (Free Resources, 1 Weekend)
You don't need a cert for these, but you need to understand them to break them:
- **Embeddings** - how text gets converted to numbers for semantic search
- **Vector databases** - Pinecone, pgvector, Weaviate (where embeddings are stored)
- **RAG pattern** - retrieve docs, stuff into prompt, get answer
- **Vercel AI SDK** - already using Next.js, this is a natural fit

Free resources:
- Vercel AI SDK docs (free)
- OpenAI Cookbook on GitHub (free)
- Pinecone learning center (free)
- YouTube: "RAG from scratch" by LangChain (free)

The goal: learn how these work so you can find vulnerabilities in them, not to build another chatbot.

### Scrimba AI Engineer Path (Already Purchased)
- Already paid for, so go through it to fill knowledge gaps on embeddings, vectors, RAG, and Vercel AI SDK
- Use it as learning material, NOT as a resume credential
- The certificate of completion has no weight with hiring managers
- Goal: understand how these systems work so you can exploit them in security contexts
- Low priority - fit it in around the real certs above
